# ✏️特别说明
---

- 为了防止脚本解析接口被盗用，使用解析功能时需要先关注公众号领取流量卡密，每日可使用100GB流量（不限制次数）
- 脚本并不会保存你的帐号cookies，也不会保存你的提取码/解析链接
- 无视黑号/免SVIP会员
- 但不支持批量文件及文件夹解析，不支持违规资源解析
